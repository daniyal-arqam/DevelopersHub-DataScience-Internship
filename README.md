# DevelopersHub Corporation — Data Science & Analytics Internship

## Intern Information
- **Program:** Data Science & Analytics Internship
- **Organization:** DevelopersHub Corporation
- **Due Date:** 28 April, 2026

---

## Task 1: Iris Dataset Visualization
- **Objective:** Explore and visualize the Iris dataset
- **Tools:** Pandas, Matplotlib, Seaborn
- **Key Insight:** Petal length and width are the strongest distinguishing features

## Task 2: Credit Risk Prediction
- **Objective:** Predict loan default using classification
- **Tools:** Scikit-learn, Logistic Regression, Decision Tree
- **Key Insight:** Credit history is the most important factor

## Task 3: Customer Churn Prediction
- **Objective:** Identify customers likely to leave the bank
- **Tools:** Random Forest, Logistic Regression
- **Key Insight:** Age and balance are the top churn drivers

## Task 4: Insurance Claim Prediction
- **Objective:** Estimate insurance charges using regression
- **Tools:** Linear Regression, MAE, RMSE
- **Key Insight:** Smoking status is the strongest predictor

## Task 5: Personal Loan Acceptance
- **Objective:** Predict loan offer acceptance
- **Tools:** Logistic Regression, Decision Tree
- **Key Insight:** Retired and student customers accept loans most
